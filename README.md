FireLoaclSWF
============

我修改过的UC版本地视频网站播放器（Firefox专用）打包版，请解压swf.rar到Profiles\chrome\后食用。

支持的网站：youku ku6 iqiyi tudou letv pplive sohu pps 17173 duowan。

UC脚本修改自cinhoo，SWF播放器采用15536900@kafan的，PPTV/17173播放器采用catcat520@kafan的。
